# Instructions

To rename a bro, got to it's line and change the value assign to the variable `"name"`.  
`subtitle1` and `subtitle2` are for the cutscenes.  

**NEVER REMOVE A BRO OR ANY OF IT'S VARIABLE** 

Example:
```json
{
    "Rambro": { 
        "name": "I must shoot",
        "subtitle1": "SUBTITLE",
        "subtitle2": "Another Subtitle"
    }
    [...]
}
